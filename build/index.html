We have developed a special software application, (“RAMSIDERHookah APP”) to control the RAMSIDER Hookahs, operating on virtually all smart phone devices via ANDROID and iOS operating systems.

The RAMSIDER Hookah APP does not require a user to register personal data, and is available for use by anyone legal to consume tobacco, (according to local laws). Once downloaded, upon starting the app, pairing between hookahs and smartphones is automatic via a secure connection. To connect to the hookah a pin code is required, and will only operate with a valid pin code. Users are prevented from being logged into more than one hookah at a time, the prevention of unauthorizedaccess to another hookah is also part of the integrated software.

The main screen features the temperature control for heating the tobacco prior to beginning the smoking session. The RAMSIDER Hookah APP is designed to be user friendly. All information is visible in real time on the User Dashboard display screen.
